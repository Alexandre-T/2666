Création de personnage

Bienvenue dans l'outil de création de vos personnages !

Bienvenue dans cet outil de création de personnage qui, en sept étapes, va vous guider dans l'élaboration de votre personnage. À chacune de ces étapes, votre personnage se peaufinera, des conseils vous seront donnés, le background de « Rome, 3000 après Caesar » vous sera détaillé. Voici la description de ces sept étapes !

  1. Première étape : Choix de la nature de votre personnage : Humain ou androïde ?
  2. Seconde étape : Description générale : vous entrerez vos noms, âge, avatar, sexe, etc…
  3. Troisième étape : Vous décrirez le passé de votre personnage, ses croyances,
  4. Quatrième étape : Vous décrirez le caractère de votre personnage,
  5. Cinquième étape : Vous choisirez les pouvoirs, les dons, les capacités uniques de votre personnage,
  6. Sixième étape : Vous pourrez prévisualiser votre fiche de personnage et décider de reprendre ou non certaines parties.
  7. Septième étape : Quand vous serez satisfait de votre personnage, que vous aurez appliqué toutes les dernières retouches, vous pourrez demander la validation de votre personnage. Dès lors, vous ne pourrez plus modifier votre fiche jusqu'à ce que le staff l'ait validé ou vous ait remonté ses retours !

Au travers de ces étapes, vous allez être accompagné pour la création de votre personnage. Le synopsys du jeu et le background sont des sources d'inspirations. N'hésitez pas à les consulter ! Vous trouverez également des idées en consultant les fiches de personnages déjà existants et les personnages prédéfinis.
Lorsque vous vous sentez prêt, lancez-vous !
